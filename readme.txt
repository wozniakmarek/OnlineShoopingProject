Witam,

Instrukcja do poprawnego działania programu:

1. Pobrać z powyższego repozytorium w wygodnej dla siebie formie.
2. Znajduję się tam plik "dbMyOnlineShopping.sql".
    Zaimportować polecenia na własnym Microsoft SQL Server
3. Zmień nazwę serwera na własną nazwę serwera w Web.config dla połączenia z bazą danych.
4. Uruchom.
